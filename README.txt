**Описание:**
Приложение для планирования тренировок с возможностью добавления, фильтрации и сохранения данных в формате JSON.

**Примеры использования:**
1. Введите дату (ГГГГ-ММ-ДД), тип тренировки и длительность.
2. Нажмите «Добавить тренировку».
3. Используйте фильтрацию по типу и дате.
4. Сохраняйте и загружайте данные через JSON.
5. Все изменения фиксируйте в Git.
